# Subharjun Bose - 3D Portfolio Showcase

An interactive, high-performance web portfolio built with **Next.js**, **Three.js**, and **Framer Motion**. This project showcases deep 3D interaction, smooth animations, and a curated selection of full-stack AI engineering projects.

## 🚀 Experience the Interaction

- **3D Hero Scene**: Immersive background with custom typography.
- **Dynamic Projects Section**: A native horizontal scroll container with snap-to-center physics.
- **Internal Vertical Scroll**: View detailed project screenshots directly inside each project card.
- **Performance Optimized**: Built with Next.js Turbopack for lightning-fast development and deployment.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [React Three Fiber](https://r3f.docs.pmnd.rs/) & [Drei](https://github.com/pmndrs/drei)
- **Styling**: SCSS Modules & Tailwind CSS
- **Interactions**: [Lenis Smooth Scroll](https://lenis.darkroom.engineering/)

## 📂 Projects Featured

*   **SafeDrive AI**: Real-time driver wellness monitor (Groq Vision).
*   **Career Mentor**: AI-powered personalized educational advisor.
*   **Resume Builder (CraftCV)**: Professional MERN stack resume platform.
*   **AutomatOS**: Llama-3 driven Windows automation kernel.
*   **Village Gentle**: Rural development AI for crop health.
